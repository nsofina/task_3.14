# Другие команды GIT

    Команды перечислены без синтаксиса.

 Просмотр изменений:

> git diff

 Кто написал строку:

> git blame

Временное сохранение изменений без коммита:

> git stash

Теги:

> git tag

Удаление файла:

> git rm

 Копирование коммита/перенос коммитов:

> git cherry-pick

 Перемещение/переименование файлов:

> git mv

История изменения указателей:

> git reflog

"Перенос" ветки: 

> git rebase
